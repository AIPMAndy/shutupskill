# shutupskill

让 AI 不废话 — 通过提示词优化改善输出风格。

## 功能

通过在 agent 的 SOUL.md 中注入优化规则，引导 AI 改变输出风格：
- 先做后说，减少询问
- 工具并行调用提示
- 错误重试引导
- 极简输出规范

## 安装

```bash
openclaw skill install https://github.com/你的用户名/shutupskill
```

## 使用

```bash
/shutup --all              # 优化所有
/shutup --agent dev        # 优化指定
/shutup --template-only    # 查看模板
/shutup --restore          # 恢复备份
```

## 工作原理

这是一个**提示词注入工具**，通过在 `~/.openclaw/agents/<agent>/workspace/SOUL.md` 末尾追加优化规则来改变 AI 行为。

**能做到的**：
- ✅ 改变输出风格（少废话）
- ✅ 引导工作流程（先做后说）
- ✅ 提供检查清单

**做不到的**：
- ❌ 保证任务完成率
- ❌ 强制工具精准度
- ❌ 让弱模型变成强模型

**效果取决于模型本身的能力。**

## 效果

### 优化前
```
"好的，让我先读取文件..."
"您希望优先优化哪个？"
```

### 优化后
```
[直接执行]
"发现 3 个问题，优化中。"
"完成。响应时间降 60%。"
```

## 输出规则

**删除：**
- 冠词、语气填充词、客套话、犹豫性表达

**允许：**
- 短句、碎片句

**优先短同义词：**
- 「大」不是「庞大」
- 「修」不是「实施解决方案」

**保持精确：**
- 技术术语
- 代码块
- 报错信息

**推荐句式：**
```
[问题][动作][原因]。[下一步]。
```
